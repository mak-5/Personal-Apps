# iOS Native Setup — Capacitor + Siri + Sideloading
**No App Store. Installed directly from your Mac to iPhone/iPad.**

---

## Prerequisites

| Tool | Where to get it | Time |
|---|---|---|
| Xcode | Mac App Store (free) | ~20 min download |
| Apple Developer Account | developer.apple.com ($99/yr) | 5 min |
| Node 18+ | Already installed | — |

---

## Step 1 — Initialize Capacitor (one time)

```bash
cd akshay-apps
npm install

# Initialize Capacitor (uses values from capacitor.config.ts)
npx cap init

# Add the iOS platform — creates the /ios folder
npx cap add ios
```

This generates an `ios/` folder — a full Xcode project that wraps your React app.

---

## Step 2 — Replace AppDelegate.swift

This enables Siri shortcuts and deep linking.

```bash
# Copy our pre-written AppDelegate
cp ios-native/AppDelegate.swift ios/App/App/AppDelegate.swift
```

---

## Step 3 — Add URL scheme in Xcode

Siri needs a URL scheme (`akshay-apps://`) to open specific screens.

1. Open Xcode: `npx cap open ios`
2. Click **App** in the left sidebar → select the **App** target
3. Go to **Info** tab → expand **URL Types**
4. Click **+** → set:
   - **Identifier**: `com.akshay.apps`
   - **URL Schemes**: `akshay-apps`

---

## Step 4 — Sign the app with your Developer account

In Xcode:
1. Click **App** target → **Signing & Capabilities**
2. Check **Automatically manage signing**
3. Set **Team** to your Apple Developer account
4. **Bundle Identifier**: `com.akshay.apps` (must be unique — add your name if needed)

---

## Step 5 — Build and install on your device

Connect your iPhone/iPad via USB (or use WiFi pairing in Xcode).

```bash
# One command builds React + syncs to Xcode + opens Xcode
npm run ios
```

In Xcode:
1. Select your device from the top device picker
2. Press **▶ Run** (or Cmd+R)
3. First time: on your iPhone go to **Settings → General → VPN & Device Management** → trust your developer certificate

App installs and launches. Done.

---

## Day-to-day workflow

**Updating the app** (no USB needed after first install):
```bash
npm run ios   # builds + syncs
# Then hit Run in Xcode (Cmd+R) — deploys over WiFi
```

**Just updating React code** (no native changes):
```bash
npm run build:native && npx cap sync
# Then Cmd+R in Xcode
```

**GitHub Pages still works too** — `npm run build` (no BUILD_TARGET) → push → auto-deploys.

## How Siri works after this

### Automatic shortcut learning
The app donates shortcuts to Siri automatically when you use each screen.
After visiting FlightScan a few times, Siri will start suggesting it proactively
on your lock screen and in Spotlight — no setup needed.

### Manual Siri phrase (best for reliability)
1. **Settings → Siri & Search → Shortcuts**
2. Find "Check flight deals" or "Check my finances"
3. Tap **+** → record your phrase: *"flight deals"* or *"my budget"*

From then on: **"Hey Siri, flight deals"** → opens FlightScan directly.

### What Siri can do
| Command | Result |
|---|---|
| "Hey Siri, flight deals" | Opens FlightScan |
| "Hey Siri, my budget" | Opens FinanceOS |
| "Hey Siri, Akshay Apps" | Opens the launcher |
| Spotlight: type "flight" | Suggests FlightScan |

## Re-signing (Developer account = once per year)

With the $99/yr Apple Developer account, your sideloaded app is valid for **1 year**.
When it expires, just hit Cmd+R in Xcode again — takes 30 seconds.

## Troubleshooting

| Problem | Fix |
|---|---|
| "No signing certificate" | Xcode → Signing → select your Apple ID team |
| App crashes on launch | Check Xcode console; usually a missing native plugin |
| Siri shortcut not appearing | Use the app a few times; iOS needs usage signals |
| "Untrusted developer" on iPhone | Settings → General → VPN & Device Management → trust |
| White screen in app | Check `base: './'` in vite.config when BUILD_TARGET=native |
